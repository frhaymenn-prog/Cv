# ✦ CV Maker — صانع السيرة الذاتية

> تطبيق ويب تقدمي (PWA) لإنشاء سيرة ذاتية احترافية باللغة العربية — يعمل بدون إنترنت ويمكن تثبيته على الجوال.

![CV Maker Preview](icons/icon-512.png)

---

## 🚀 المميزات

| الميزة | الوصف |
|--------|-------|
| 🎨 **3 قوالب** | كلاسيك · عصري · جانبي |
| 🎨 **6 ألوان** | تخصيص لون السيرة |
| 📷 **صورة شخصية** | رفع وعرض الصورة |
| 💼 **خبرات متعددة** | إضافة وحذف خبرات بلا حدود |
| 🎓 **تعليم متعدد** | إضافة وحذف مؤهلات |
| 🌐 **بدون إنترنت** | PWA تعمل Offline |
| 💾 **حفظ تلقائي** | البيانات تُحفظ في المتصفح |
| 📄 **تحميل PDF** | طباعة مباشرة كـ PDF |
| 📱 **متجاوب** | يعمل على الجوال والكمبيوتر |
| 🔒 **خصوصية كاملة** | لا يُرسَل أي بيانات للخارج |

```
cv-app/
```

## 🛠 طريقة التشغيل المحلي

```bash
# 1. استنساخ المشروع
git clone https://github.com/username/cv-maker.git
cd cv-maker

# 2. تشغيل سيرفر محلي بسيط
npx serve .
# أو
python3 -m http.server 8080

# 3. افتح المتصفح على
http://localhost:8080
```

> ملاحظة: يجب تشغيل التطبيق عبر سيرفر (وليس مباشرة بفتح index.html) حتى يعمل الـ Service Worker.

## 🌐 النشر على Netlify

1. ارفع المشروع على GitHub
2. سجّل دخولك على [netlify.com](https://netlify.com)
3. اختر **"Import from Git"**
4. اختر مستودعك → اضغط **Deploy**
5. الموقع سيكون جاهزاً في أقل من دقيقة ✅

---

## 📱 تحويل لتطبيق أندرويد عبر WebIntoApp

1. افتح [webintoapp.com](https://webintoapp.com)
2. أدخل رابط Netlify الخاص بك
3. ضع اسم التطبيق: **CV Maker**
4. ارفع الأيقونة من `icons/icon-512.png`
5. اضغط **Generate App**
6. حمّل ملف الـ APK ✅
